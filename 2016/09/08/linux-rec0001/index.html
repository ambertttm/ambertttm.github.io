Linux记录-(1)

Some tricks when using Linux OS.

目录与文件数目统计

统计当前文件夹下文件的数目

ls -l | grep "^-" | wc -l

统计当前文件夹下目录的数目

ls -l | grep "^d" | wc -l

统计当前文件夹下文件的个数，包括子文件夹内的

ls -lR | grep "^-" | wc -l

统计文件夹下目录的个数，包括子文件夹内的

ls -lR | grep "^d" | wc -l

说明：ls -l表示输出当前路径下所有目录和文件的信息，grep "^-"把长列表的输出信息过滤一部分，安装定好的规则保留，wc -l统计输出信息的行数。

解决删除时Device or resource busy的问题

执行：

lsof +D /home/to/target/path

在屏幕输出中得到PID，执行：

kill -s 9 pid-number

之后重新删除。
